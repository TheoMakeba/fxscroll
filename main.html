Mise en place

Pour utliser fxscroll, vous devez premièrement commencer par télécharger la libraire et inclure les fichier CSS et JavaScript dans votre document HTML de manière traditionnelle.
// Import du fichier CSS et JavaScript

Utilisation

Pour utiliser fxscroll vous devez d'anore créer une instance de fxscroll puis dde faire usage des méthodes de cette instance pour cibler des éléments et spécifier certains paramètres d'animation.
// Création de l'objet fxscroll let fxscrollInstance = new fxscroll();
L'objet crée possèdenune seule méthode avec laquelle il sera possible de cibler et de de spécifier les parmètre d'animation.

Cette méthode est : createContext()
// createContext let fxscrollInstance = new fxscroll(); fxscrollInstance.createContext({ 'context' : [ { target : ["#myId", document.getElementById("myId2"), "[class='myClass']"], options : { ratio : 0.2 // Spécifie le ratio de visibilité des éléments ciblés : comprit entre 0 et 1 flow : "r-t" // Un mot clé de l'effet d'animation clear : true|false // Un booléen qui spécifie si l'élement ciblé doit être masqué si il n'est plus visible } } ] })
Cette méthode prend en argument un objet JavaScript qui doit necessairement posséder une propriété nommée : context .

Cette propriété doit avoir pour valeur un tableau, ce dernier doit pouvoir contenir des objets JavaScript qui répresentent un ensemble d'éléments à cibler ainsi que leurs options spécifiques.

Les objets JavaScript contenu dans le tabeleau du champ context doivent spécifier les champs ci-après :

  • 1. target

    Cette propriété est un tableau qui contient soit des sélécteurs sous forme de chaîne de caractère, soit des noeud du DOM.
    C'est sur ces éléments que porterons les effets d'animations qui seront spécifiés par le champ suivant

  • 2. options

    Cette propriété est un objet JavaScript dont on doit spécifier les propriétés ci-après :

    • A. ratio

      Cette propriété est un entier flottant qui spécifie le ratio de visibilité, c'est à dire un entier compris entre 0 et 1 qui indique la hauteur minimum que doit avoir un objet par rapport au bas de l'écran pour qu'une animation d'apparition soit déclanchée.

    • B. flow

      Cette propriété est une chaîne de caractère qui spécifie le type d'animation à appliquer.
      Il faudrait savoir qu'il exite 5 effets de base disponible :

      t-b

      Pour top-bottom, il applique, sur un élément ciblé, un effet de glissement doux de haut vers le bas

      b-t

      Pour bottom-top, il applique, sur un élément ciblé, un effet de glissement doux du bas vers le haut

      l-r

      Pour left-right, il applique, sur un élément ciblé, un effet de glissement doux de la gauche vers la droite

      r-l

      Pour top to bottom, il applique, sur un élément ciblé, un effet de glissement doux de la droite vers la gauche

      scale-in

      Pour top to bottom, il applique, sur un élément ciblé, un effet de zoom sortant donnant la sensation q'un élément devient progressivement grand jusqu'à atteindre sa taille normale.

    • C. clear

      Cette propriété est un booléen qui spécifie si oui ou non un élément douit être masqué de nouveau s'il nest plus visible dans la fenêtre du navigateur

Il faut aussi savoir qu'il est impératif d'appeller la méthode init() sur l'instace de fxscroll pour commencer le traitement, sinon rien ne se passera visuellement.
var ref = new fxscroll(); ref.createContext({ "context" : [ target : ['div[id="myId"]'], options : { ratio : 0.3, flow : "r-l", clear : true }, target : ['.myClassName'], options : { ratio : 0.1, flow : "scale-in", clear : false }, target : ['#idRoot', document.getElementById("myId")], options : { ratio : 0.8, flow : "b-t", clear : true } ] }); ref.init(); // pour initier les effets

Quelques precautions à prendre

Etant donné que le coeur du calcul est basé sur la hauteur des éléments ciblés, il faudrait prendre soin de ne pas cibler les éléments dont la hauteur est trop grande que la celle du navigateur.

Il est aussi recommandé de ne pas cibler deux fois les mêmes éléments, ceci pourrait avoir des efeets indesirables.
